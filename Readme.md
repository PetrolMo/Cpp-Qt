# Readme

这是物联网的中期项目——遍历传感器矩阵

使用方法是选择一个点作为起点(默认第一个被选择的点为起点)，遍历余下选择的点，然后点击开始计算，程序会选择一条时间最短的路径以及一条距离最短的路径进行展示。同时会显示整个计算所花费的时间和内存。
